Запуск программы:
Необходимо запустить скрипт main.py с параметрами "профессия" "количество страниц", например:

python main.py программист 6

Результатом работы будет служить коллекция "профессия" в базе данных vacancy 

Для поиска максимальной зп используется формат "макс" "профессия" "зп", например:

python main.py макс программист 15000
